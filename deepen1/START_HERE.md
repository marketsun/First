## 🎯 첫 검색 해보기

1. 터미널에서 `python app.py` 실행
2. 브라우저에서 `http://localhost:5000` 접속
3. 검색창에 키워드 입력 (예: "맛집", "여행", "브랜드명")
4. "검색" 버튼 클릭
5. 1-2분 대기 (진행 상황 표시됨)
6. 결과 확인!

## 🎨 주요 화면

### 메인 화면 (/)
- 키워드 검색
- 진행 상황 표시
- 사용 방법 안내

### 결과 화면 (/results)
- 구글 검색 결과 (상위 50개)
- 유튜브 검색 결과 (상위 50개)

### 검색 기록 (/history)
- 저장된 모든 검색 기록
- 결과 재확인
- 기록 삭제

## 🐛 문제 발생 시

### 1. 패키지 설치 오류
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### 2. 포트 충돌 (Address already in use)
`app.py` 마지막 줄 수정:
```python
app.run(debug=True, host='0.0.0.0', port=5001)
```

### 3. ChromeDriver 오류
- Chrome 브라우저 최신 버전 설치
- 프로그램 재실행 (자동 해결)

### 4. 더 자세한 해결 방법
`INSTALL.md` 문서의 "문제 해결" 섹션 참고
